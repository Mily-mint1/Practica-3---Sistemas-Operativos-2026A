ESCUELA POLITECNICA NACIONAL

Integrantes: Emily Morales, Vanessa Paucar, Karen Tapia, Daniel Tixicuro, Emily Vargas
Paralelo: GR2CD
Fecha de Publicación: 12/05/2026

Objetivos:
1. Comprender la gestion de proces en Linux y conocer sus funciones principales: fork(), system y exec para la creacion y control de procesos en Linux.
2. Implementar jerarquias de procesos padre e hijos y comprender cual es su orden de ejecucion para conocer los distintos casos que se puedan presentar.
3. Aplicar funciones importantes para una mejor gestion de procesos jerarquicos de padre e hijo.

Descripción: Este repositorio contiene el desarrollo completo de la Práctica 3 del Laboratorio de Sistemas Operativos. 
El objetivo principal de este trabajo fue explorar y aplicar de manera práctica los mecanismos de creación y gestión de procesos en un entorno Linux.
A lo largo de la práctica, se implementaron diversos programas en lenguaje C que demuestran el uso de llamadas al sistema fundamentales como fork() 
para la clonación de procesos, exec() para la ejecución de nuevos programas y system() para comandos del sistema. El proyecto pone especial énfasis 
en la resolución de problemas de sincronización, la identificación de PID y la gestión de jerarquías para evitar estados de procesos huerfanos en los hijos.
Toda la implementación se encuentra organizada por carpetas siguiendo la estructura requerida, incluyendo los códigos fuente y la documentación 
de las pruebas realizadas en una máquina virtual Ubuntu.
